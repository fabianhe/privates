# Privates

Private (fund) investment modeling tools.

## Key Development Tools

## [uv](https://docs.astral.sh/uv/)
> An extremely fast Python package and project manager, written in Rust.

Please manage your Python version as well as your virtual environment with uv.

You can activate the virtual environment as follows:
```bash
source .venv/bin/activate
```

## [pre-commit](https://pre-commit.com/)
> A framework for managing and maintaining multi-language pre-commit hooks.

Please intall this once in your virtual environment as follows:

```bash
pre-commit install
```

Within your virtual environment, you can run the pre-commit checks on all files as follows:

```bash
pre-commit run --all-files
```
